# database-programming-starter-repo

Dette starter repo indeholder alt hvad I skal bruge til at komme i gang med MongoDB.

I mappen w3schools-mongodb-tutorial ligger en fil stub.js, som I kan bygge videre på.

I skal udbygge stubben, så den indeholder eksempler på alle operationer fra W3Schools MongoDB tutorial'en.

I kan finde tutorialen her: https://www.w3schools.com/mongodb/index.php
